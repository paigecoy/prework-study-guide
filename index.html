Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Metadata tags go inside the head.
  • The title element defines the title for a webpage; is required and is extremely important for Search Engine Optimization (SEO).
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a website.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata (information used by the browser) such as the character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • A declaration contains CSS property we want to apply and the value of the property.
  • The internal style sheet contains CSS rules for the webpage in the head section of the HTML file; rules apply to all tagged page elements; possible to embed an entire stylesheet directly in the HTML file using a stlye element in the head element.
  • The external CSS style sheet is the most common way to style a webpage because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier using the link element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A tells Git to stage all files
  • git commit -m "messageaboutchanges" commits changes with message of what specifically was changed
  • git pull receives branch's modifications into local environment to ensure there are no conflicts between GitHub and local repo
  • git push pushes local changes to remote GitHub branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A string should be enclosed in single or double quotation marks.
  • Numbers don't need quotes around them.
  • Boolean is a true/false value and those keywords don't need quotes.
  • An array allows you to store multiple values in a single reference.
